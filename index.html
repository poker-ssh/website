Poker-Over-SSH

Multiplayer Poker on terminal

This is Poker built with Python which you can access through your terminal and play with others using SSH


TO PLAY:

Copy paste in terminal/command prompt

ssh play.poker.qincai.xyz
Or if you don't want to generate a keypair, use: ssh guest@play.poker.qincai.xyz
